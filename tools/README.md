# Actuator Debugging Tool / 关节电机调试工具

This directory contains the debugging tool and documentation for onero actuator modules (XD5746 / XD5757). This tool is for actuator-level debugging only and is **not** intended for robotic arm control.

本目录包含 onero 关节模组（XD5746 / XD5757）的调试工具及使用说明。该工具仅用于关节电机调试，**不适用于**机械臂控制。

## Files / 文件说明

| File | Description |
|------|-------------|
| `motor_debug_tool_v1.0.2.zip` | Debug software / 调试上位机软件 |
| `Debug_Tool_Quick_Guide.pdf` | Quick guide (English) |
| `调试工具使用说明.pdf` | 使用说明（中文） |

## Connection / 连接方式

Connect the actuator to your PC via the USB-to-CAN adapter included with the product.

使用产品附带的 USB-to-CAN 适配器将关节电机连接至 PC。
